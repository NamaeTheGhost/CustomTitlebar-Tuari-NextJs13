## Getting Started

Download Project

```bash
git clone https://github.com/NamaeTheGhost/CustomTitlebar-Tuari-NextJs13.git

#and

cd CustomTitlebar-Tuari-NextJs13

#and

npm install

#and

npm run tauri dev
```
